# 季節と天気のアンビエント

四季 × 天気 × 時間帯 × 環境音 を選び、ただ眺めて心を整えるアンビエント・アプリ。
描画はシェーダー（GLSL）で「現象」を計算生成する。

> 現在は **土台（配管）段階**。WebGL が動くことを確認するための最小画面（色がゆっくり移ろうだけ）のみ。
> 本格的な情景（最初の見本＝夏・雨・夕方・ヒグラシ）はこれから作り込む。

## 技術スタック

- **描画**: 生 WebGL によるフルスクリーン・シェーダー構成
- **ビルド**: Vite
- **ホスティング**: GitHub Pages（`main` への push で自動デプロイ）

### なぜ Three.js ではなく軽量シェーダー構成か

このアプリの主役は「画面いっぱいの 1 枚絵をフラグメントシェーダーで描く」こと。
生 WebGL なら依存ゼロでバンドルが最小、スマホでの起動も軽く、GLSL を直接握れて
雨粒の屈折などを最大限作り込める。多重パス（FBO）の管理が複雑化した段階で
Three.js の採用を改めて検討する。

## 開発

```bash
npm install     # 初回のみ
npm run dev     # ローカル開発サーバ
npm run build   # 本番ビルド（dist/ を生成）
npm run preview # ビルド結果のプレビュー
```
